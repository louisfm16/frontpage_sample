# Tagrem Test Site by Louis Medina
## What I used
`
 - JQuery https://jquery.com/
 - Flexbox Grid http://flexboxgrid.com/
 - SCSS
 - Gulp
 - Node (for dev use only)
`

## How to run in Node:
`
npm init
npm install
`

## Note may need gulp & gulp-cli):
`
npm install gulp --save-dev
npm install gulp-cli -g
`

## Live reload
`
gulp watch
`
